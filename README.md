# mycompose

**How to implement**

> add the JitPack repository to your settings.gradle file 
```
repositories {
    ...
    maven { url 'https://jitpack.io' }
}
```
> add dependencies to your app build.gradle file
```
dependencies {
    ...
	implementation 'com.github.hoangdinhhiep:mycompose:{{TAG}}'
}
```
